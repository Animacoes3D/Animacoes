Sobre o Projeto

Esse site é resultante do Projeto PIBIC - Jr, N° 002/2023/DPI/PPGI/IFAM/IC, utilizando o programa Blender 4.3, desenvolvido com o apoio e incentivo do Conselho Nacional de Desenvolvimento Científico e Tecnológico (CNPq) e do Instituto Federal do Amazonas (IFAM) Campus Tefé.

Objetivo

Desenvolver recursos visuais para facilitar o entendimento de conceitos abstratos em química, utilizando animações 3D criadas com Blender. O projeto visa tornar o aprendizado mais dinâmico e acessível para estudantes do ensino médio e superior.

Metodologia

Utilizamos o software Blender para criar modelos moleculares e cenas animadas que representam processos químicos complexos. Cada animação é cuidadosamente planejada para garantir precisão científica e clareza visual, seguida da produção de vídeos educativos para o YouTube.

Aplicações

Os materiais produzidos podem ser utilizados em aulas de química, tanto no ensino presencial quanto no remoto. As animações servem como ferramentas complementares que ajudam os estudantes a visualizar conceitos que são difíceis de compreender apenas com explicações teóricas.
